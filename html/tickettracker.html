RSP Ticket Tracker Web App

RSP Ticket Tracker


RSP Ticket Tracker is a web app used to track customer issues with apps that have been developed by Random Sin Productions. It was created in Visual Studios with the C# .net framework, MVC, Azure's SQL database, and Azure web hosting. SendGrid was integrated to validate new users through a link in their provided email address. I also implemented a one click Google OAth2 login, which allows users to quickly setup a validated account with their currently logged in Google profile. A validated account can open as many tickets as they like, but only administrative users can delete, edit, and mark tickets as resolved. RSP Ticket Tracker is still in production, but there is currently an imperfect version on the web at, RSP Ticket Tracker.
