Happy Feet Web App

Happy Feet


Happy Feet is a mock ecommerse shoe store that I created with three fellow computer scientists. We each did separate parts of the web app and then worked together to combine them. I worked on the checkout page in Eclipse, using Java, Java Servlets, HTML, and SQL. My focus was on both the client side validation using JS, the server side validation, and additional database security. AJAX was used to check the database for the given zip code and automatically fills in the city, state, and tax information in the form. I setup the site in Amazon's AWS Ubuntu instance running Tomcat and MYSql. You can checkout the site by clicking this link, Happy Feet.
