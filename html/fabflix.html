Fabflix Webapp


Fabflix is a mock ecommerse website that sells dvd's which are stored in a MySql database, in an AWS instance running Tomcat, and the web app was designed in Eclipse using Java Servlets along with JSP's, AJAX, JS, CSS, and HTML. This is the first web project that I have worked on, so there are many flaws, such as commenting, security and organizing of the code. Fabflix was my introduction to web based projects and I learned how to use a variety of techniques, among these are iframes, sychronous and asychronous database queries, seperating CSS and JS from HTML, using JS for dynamic web pages, setting up HTTPS, and much more. Follow the link to view the web app, Fabflix Link.

Known issue: The initial press of a link does not fetch the DVD's from the database. You have to wait a second and press the link again.
